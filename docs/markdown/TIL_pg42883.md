# 📝 TIL (Today I Learned) - 알고리즘 문제 풀이 복습

1. [PG] 42883 - 큰 수 만들기 (Java)
   핵심 아이디어: 앞자리에 큰 숫자가 올수록 전체 숫자가 커지므로, 스택(Stack)을 사용해 현재 숫자보다 작은 앞의 숫자들을 제거하며 진행 (Greedy).
   문제 해결의 키:
   루프를 돌며 stack.peekLast() < current일 때까지 pop 수행.
   지울 수 있는 횟수 k가 남아있을 때만 수행.
   숫자가 내림차순이라 k가 남은 경우(예: "999", k=1), 뒤에서부터 남은 k만큼 잘라내기.
2. Java 코드 최적화 팁 (Performance Up! 🚀)
   무거운 형변환 피하기:
   Integer.parseInt(String.valueOf(char))는 매번 객체를 생성하므로 느림.
   char 타입 그대로 비교하거나, 숫자가 필요하면 c - '0' 산술 연산 사용.
   자료구조 선택:
   ArrayDeque는 스택/큐 양방향 활용 시 좋지만, 극강의 성능이 필요할 땐 char[] 배열과 포인터(top) 조합이 가장 빠름.
   결과 생성:
   StringBuilder에 하나씩 넣는 것보다 new String(charArray, 0, length) 방식이 더 효율적일 수 있음.
3. Python 꿀팁 복습
   내림차순 반복문:
   reversed(range(n)) : 가독성 좋음 (추천).
   range(start, end, -1) : 시작/끝 범위를 정교하게 조절할 때 사용.
   리스트 컴프리헨션(List Comprehension):
   [x for x in list if 조건] : 필터링 (뒤에 if).
   [A if 조건 else B for x in list] : 값 치환 (앞에 if-else).
